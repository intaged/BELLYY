# Belly Inflation Game

A fun web-based game built with HTML5 Canvas and JavaScript.

## Features
- Interactive gameplay
- Custom cursor effects
- Social media integration
- Responsive design

## Setup
1. Clone the repository
2. Open index.html in your browser
3. Start playing!

## Technologies Used

- CSS3
- JavaScript
- Canvas API
